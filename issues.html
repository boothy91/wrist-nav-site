Known Issues & Upcoming Features

⚠️ These are known issues currently under investigation. Behaviour may vary depending on device, OS version, and usage.
🐞

Current Issues

Zoom buttons on map

  • Limited screen space on watch displays makes placement difficult
  • Conflicts with existing gesture controls (pinch, crown, swipe)
Status: Investigating UI approaches

Strava disconnecting

  • Users report Strava sessions unlinking unexpectedly
  • May require re-authentication between uploads
  • Potential API/session persistence issue
Status: Investigating stability and auth flow

Multi upload to Plan tab

  • Uploading multiple GPX files at once is not fully supported
  • Files may fail to appear or only partially import
Status: Investigating batch import handling
🛠️

Planned

Auto-detect map region

  • Automatically suggest or prioritise map downloads based on current location
  • Reduces friction when setting up maps on-device

Haptic navigation

  • Vibration alerts when going off-route
  • Advance warning for upcoming turns

Health Connect integration

  • Sync workouts and health data with Android Health Connect
  • Improves compatibility with other fitness apps

Final workout summary

  • Post-activity screen with key stats
  • Distance, time, elevation, pace and summary metrics
