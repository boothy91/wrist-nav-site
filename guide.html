Home Screen

  • Start, pause and stop recording from the home screen
  • GPS status indicator — red (no fix), amber (weak), green (good)
  • Enable Nav mode to follow a GPX route — pressing start with Nav on goes straight to the home map page
  • Swipe from Home to open the map quickly, or swipe through Stats pages to reach the stats map
🗺️

Map Screen

Controls

  • Pinch to zoom, swipe to pan
  • Crown / rotary input support is available for compatible watches, including OnePlus Watch 3 bezel support
  • Long press to open the main menu
  • Location button — red = locked, map follows your GPS position · blue = free, moveable map
  • Compass — tap to cycle modes · drag right to go back to stats pages
  • Map Lock — tap the clock/battery bar to fully lock the map from all motion, tap again to unlock · small dot shows lock state

Compass Modes

  • 🔴 Red — North locked: Map always faces north, compass tape shows N
  • 🟢 Green — Heading mode: Map rotates with your direction of travel
  • 🔵 Blue — Free compass: Map locked north, compass tape moves live with your bearing

Navigation Overlay

  • Remaining distance to end of route (toggle in Map Settings → Nav Distance)
  • Live elevation profile with your position and remaining climb (toggle in Map Settings → Nav Elevation)

Bezel Stats

  • Shows live workout data curved along the bottom of round watch screens
  • Straight bar on square watches
  • 3 configurable stat slots — enable and configure in Map Settings → Bezel Stats
  • Colour controlled by Map Text colour setting

Minimal Map Mode

  • Strips the map to roads, paths and water only
  • Pure black background — maximises battery on AMOLED displays
  • Select via Settings → Map Theme
⚠️ Map lock requires swiping between pages to activate.
📥

Maps Page

Manage Maps

  • Tap to open already downloaded maps
  • Tap any map to delete it

Download Maps

  • Browse regions and download directly to the watch — no phone needed
  • Some countries have a single map file, others use leaf maps by region
  • Tap any region to download
  • Available regions: boothy91.github.io/wrist-nav-maps
🗂️

Tracks Page

Import GPX

  • Import GPX files from any file manager, Google Drive or email
  • Download OSM traces directly to the watch

Track List

  • Each track shows name and file size
  • Tap a track to open its detail view
  • Toggle visibility — doubles as the active navigation route selector
  • Delete tracks

Track Detail — 3 Pages

  • Page 1: Distance and date
  • Page 2: Elevation details
  • Page 3: Elevation profile
📊

Stats Pages

Customisable Stats

  • Up to 10 pages, 3 or 6 stat slots per page
  • Tap 3 or 6 either side of the circle in Stats Edit to switch slot count
  • Text scales automatically to fit
  • Configure in Settings → Stats Pages
  • Add and remove pages as needed

Recording

  • Recording continues when the screen sleeps via a background service
  • Vibrates every km or mile
  • Lap splits with vibration and split time display
  • Optional countdown timer before recording starts (3s or 5s)
  • Adjustable minimum GPS recording distance and time interval
  • Activity type written into GPX — Run, Trail Run, Walk, Hike, Road Cycle, Mountain Bike, Gravel, Kayak, Ski, Snowboard
  • GPX exported with timestamps, elevation, heart rate and cadence data
⌚

Watch Face Complications

Assign any WristNav stat to a complication slot on your watch face. Updates live during recording.
  • 15 complication providers available — one per stat
  • Select from your watch face customisation screen
  • Tap any complication to open WristNav
  • Shows live data during recording, placeholder when not recording

Phone Companion App

The watch app works fully standalone — the phone app is optional.

Send Tab

  • Send GPX route files directly to the watch
  • Send .map files to the watch for offline use
  • Log in to Strava to sync recorded activities

Received Tab

  • View GPX files recorded on the watch and transferred to your phone
  • Upload to Strava
  • Save to phone storage
  • Delete files

Plan Tab

  • Plan a route directly on the map with routed or straight-line segments
  • Load existing GPX files to view on the map
  • OSM, Topo and Satellite map styles
  • 3D terrain view
  • Export as GPX or send straight to the watch

Theme Tab

  • Create custom map themes for the watch
  • 10 colour presets to start from
  • Tap any element to change its colour with a full colour picker
  • Live map preview — see exactly what the theme looks like before sending
  • Send finished themes to the watch

Settings Tab

  • Phone app theme — Light / Fjord / Dark
  • Distance units — km / mi
  • Map style — Standard / Topo
  • Route colour and width
  • Auto delete received files — Off / 7 days / 30 days
🐛

Reporting Crashes

If the app crashes, crash reports are automatically sent to the development team via Sentry the next time the watch connects to the internet. You do not need to do anything — crashes are captured and reported automatically.
