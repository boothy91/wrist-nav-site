Home Screen

  • Tap the large circle button to start recording — a second tap goes straight to the map
  • GPS status indicator — red (no fix), green (good fix)
  • NAV OFF / NAV ON toggle — tap to pick a GPX route to follow, tap again to clear it
  • When a route is loaded, starting recording goes straight into the map flow
  • Swipe right from Home to open the map directly
🗺️

Map Screen

Controls

  • Pinch to zoom, swipe to pan
  • Crown / rotary input for zoom on compatible watches
  • Long press to open the main menu
  • Location button — red = locked to GPS · green = free pan (can change colours within settings)
  • Compass tape — tap to cycle modes · drag right to go back to Recording Screen (can swap to a zoom function or remove completely in settings, also can move to left or right hand side, this also swaps the side of the location button)
  • Map Lock — tap the clock/battery bar to lock all input

Compass Modes

  • 🔴 Red — North locked: Map always faces north
  • 🟢 Green — Heading: Map rotates with direction of travel
  • 🔵 Blue — Free compass: Map locked north, tape moves live with your bearing

Navigation Overlay

  • Remaining distance to end of route — toggle in Settings → Map → Map Settings → Nav Distance
  • Live elevation profile with current position marked — toggle in Settings → Map → Map Settings → Nav Elevation

Bezel Stats

  • Live workout data curved along the bottom of round screens, straight bar on square watches
  • 3 configurable stat slots — enable in Settings → Map → Map Settings → Bezel Stats, then swipe right to configure
  • Text colour controlled by the Map Text colour setting
📥

Maps Page

Download Maps

  • Browse regions and download directly to the watch — no phone needed
  • Some countries have a single file, others use regional leaf maps
  • Tap any region to download
  • Available regions: boothy91.github.io/wrist-nav-maps
  • Swipe through downloaded maps — tap to delete
💡 You can also send .map files directly from the phone companion app (Max 15mb).
🗂️

Tracks Page

Import GPX

  • Import GPX files sent from the phone companion app
  • Download OSM traces directly to the watch

Track List

  • Each track shows name and distance
  • Tap a track to open its detail view
  • Delete tracks from the detail view

Track Detail Pages

  • Stats: Distance, total time, moving time, date
  • Heart rate: Avg, max and min HR (watch-recorded tracks only)
  • Cadence: Avg cadence, max cadence, total steps (watch-recorded tracks only)
  • Elevation: Gain, loss, max and min elevation
  • Graph: Full elevation profile graph
📊

Stats Pages

Customisable Stats

  • Up to 10 pages, 3 stat slots per page
  • Optional stat labels — toggle via phone companion app Settings tab
  • Configure in Settings → Stats → Stats Pages
  • Add and remove pages freely

Recording

  • Recording continues when the screen sleeps via a background service
  • Vibrates at every km or mile split with split time shown on screen
  • Optional countdown timer before recording starts (3s or 5s)
  • Auto-pause when speed drops below a configurable threshold
  • Adjustable minimum GPS recording distance and time interval
  • Activity type written into GPX — Run, Trail Run, Walk, Hike, Road Cycle, Mountain Bike, Gravel, Kayak, Ski, Snowboard
  • GPX exported with timestamps, elevation, heart rate and cadence data

Phone Companion App

The watch app works fully standalone — the phone app is optional.

Send Tab

  • Send GPX route files directly to the watch
  • Send .map files for offline use
  • Log in to Strava to sync recorded activities

Received Tab

  • View GPX files recorded on the watch
  • Upload to Strava
  • Save to phone storage
  • Delete files

Plan Tab

  • Plan routes with routed or straight-line segments
  • Load existing GPX files on the map
  • OSM, Topo and Satellite styles
  • 3D terrain view
  • Export as GPX or send straight to watch

Theme Tab

  • Create custom map themes for the watch
  • 10 colour presets to start from
  • Full colour picker per element
  • Live map preview before sending

Settings Tab

  • Phone app theme — Light / Fjord / Dark
  • Distance units — km / mi
  • Map style — Standard / Topo
  • Route colour and width
  • Stats labels — show or hide stat name labels on watch stats pages
  • Auto delete received files — Off / 7 days / 30 days
  • Toggle on or off the maps and tracks menu options on the watch to clean up the UI

Reporting Crashes

Crashes are automatically captured and reported to the development team via Sentry. You don't need to do anything.
